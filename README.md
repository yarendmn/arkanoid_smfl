# Arkanoid (Breakout Clone) - C++

This project is a multi-level Arkanoid-style game developed in C++ using SFML.

## Features (Planned)

* Multi-level system (loaded from files)
* Paddle and ball mechanics
* Brick breaking system
* Power-ups (paddle expansion, ball speed)
* Lives system
* Score tracking
* Level transition screens

## Technologies

* C++
* SFML
